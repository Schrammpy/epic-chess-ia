# Narrador Épico de Partidas

Landing que convierte registros de partidas (PGN/FEN o lista corta) en crónicas épicas medievales.

## Tecnologías
- HTML + CSS modular (tokens, base, components, app)
- JavaScript vanilla (`assets/js/app.js`)
- LM Studio local (endpoint OpenAI-compatible)

## Cómo correr
1. Abrí `index.html` en el navegador.
2. Asegurate de tener **LM Studio** en `http://127.0.0.1:1234/v1` con **CORS ON**.
3. Pegá un PGN o movimientos, hacé click en **Generar relato**.

> Para publicar como sitio estático: activá **GitHub Pages** en Settings → Pages.
